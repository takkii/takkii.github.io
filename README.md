### takkii.github.io

GitHub page.

#### 環境構築

```markdown
gem install jekyll

rbenv rehash

jekyll new takkii.github.io
※ takkiiの部分はユーザ名

cd takkii.github.io

git init
... レポジトリ登録

jekyll s
jekyll build
... ローカル確認後、デフォルトブランチへ反映
```
